WARNING!!!

  • File yang diterima hanya dalam format CSV. Format lain tidak akan diterima.
  • Data yang telah diimpor tidak akan tersimpan. Jika Anda menutup atau merefresh halaman ini, semua perubahan yang telah dilakukan akan hilang.
  • WhatsApp yang digunakan adalah WhatsApp Desktop.

User Manual Data Request


  1. Masukkan Format Pesan

    Tulis format pesan yang diinginkan pada kolom "Format Pesan".

    Gunakan:


    • {1} untuk nama Client
    • {2} untuk link data request

    Contoh data yang ada pada CSV anda:


    [Nama Client, Nomor Telepon, Link Data]

    Hebert,62812xxxxxxx9,drive.google.com/hebert


    Contoh format pesan yang dapat anda buat seperti ini:

    Halo Pak/Bu {1},
    Berikut kami lampirkan link data request 2024: {2}

    Hasil yang akan terkirim:

    Halo Pak/Bu Hebert,
    Berikut kami lampirkan link data request 2024: drive.google.com/hebert

    Jika anda ingin mengubah format pesan setelah mengimport CSV, anda harus menekan tombol "ubah format".
    Jika sudah selesai mengubah tekan tombol "selesai mengubah".


  2. Impor File CSV

    Impor file CSV yang telah dibuat, format CSV yang diterima sebagai berikut:


    [Nama Client, Nomor Telepon, Link Data]

    Pastikan memberikan 1 baris kosong antara header [Nama Client, Nomor Telepon, Link Data] dan data yang akan dimasukkan.
    Data pertama dapat dimulai dari baris ke-3 (sistem tidak akan membaca data pada baris 1-2).


    Klik UNDUH untuk mengunduh contoh format CSV.


  3. Proses CSV


    Setelah mengimpor CSV, sistem akan memproses file tersebut menjadi baris-baris.
    Setiap barisnya terdiri dari Nama Client, Nomor Telepon Client, dan Tombol.


  4. Kirim Pesan

    Setelah CSV berhasil diproses, tekan tombol "Kirim Pesan" pada baris Client yang ingin dikirimkan permintaan datanya.


  5. Arahkan ke WhatsApp


    • Setelah menekan tombol, Anda akan diarahkan ke api.whatsapp.com, dan kemudian secara otomatis diarahkan ke WhatsApp Desktop.
    • Pesan akan terisi otomatis dan siap untuk dikirim.
    • Jika pesan tidak terisi otomatis, kembali ke api.whatsapp.com yang sudah terbuka (pastikan nama Client sesuai).
    • Kemudian tekan tombol "Continue to Chat" untuk diarahkan ke WhatsApp Desktop, di mana pesan akan terisi otomatis dan siap dikirim.

    Jika pesan masih belum terisi otomatis, Anda dapat mengulangi langkah ini.
